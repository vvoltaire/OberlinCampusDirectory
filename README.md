
# Oberlin Campus Directory

Centralized campus information platform for students to quickly access building hours, directions, events, and resources.

## Live Demo

[https://v0-new-project-hyzvelwh0bz.vercel.app/](https://v0-new-project-hyzvelwh0bz.vercel.app/)

## Overview

Oberlin Campus Directory is a web application designed to consolidate fragmented campus information into a single, searchable interface. It improves discoverability of building details and enhances day-to-day navigation for students.

## Features

### Campus Information Hub

* Centralized listing of campus buildings
* Includes hours, directions, and event information
* Consistent layout across 20+ locations

### Search and Discovery

* Full-text search across buildings and tags
* Filtering by categories and attributes
* Fast navigation to relevant campus resources

### Sorting and Tagging

* Sort buildings by relevance, name, or category
* Tag-based organization for improved browsing
* Flexible structure designed for future expansion

### Media Support

* Displays images and contextual building information
* Enhances clarity and usability for each listing

## Tech Stack

* JavaScript
* TypeScript
* React
* HTML / CSS

## Architecture Notes

* Frontend-first design with modular React components
* Data layer structured for future migration to centralized database
* Built with scalability in mind for campus-wide expansion
* Designed to support embedded tools and real-time updates in future iterations

## Project Goals

* Reduce time spent searching for campus information
* Improve accessibility of building-specific details
* Create a foundation for a unified campus information system

## Future Improvements

* Centralized backend database integration
* Real-time updates for hours and events
* Expanded campus-wide tool integrations
* User personalization and saved locations
